# WordPress Security Audits

Security assessments conducted on 6 live production 
WordPress websites. Each audit covers:

- SSL/TLS configuration
- Security headers analysis  
- Vulnerability scanning (WPScan)
- Malware and blacklist checks
- Manual testing (user enumeration, 
  XML-RPC, REST API exposure)
- Remediation and hardening

## Tools Used
WPScan, Qualys SSLLabs, SecurityHeaders.com, 
Wordfence, VirusTotal, MXToolbox, 
Google Safe Browsing, Cloudflare
